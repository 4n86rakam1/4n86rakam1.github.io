    There is a Volume named flag, and it contains the flag at the path flag_for_status. This Volume is mounted at /usr/share/nginx/.lemme_try_hiding_flag_with_dot_in_front. Thus, the flag is located at /usr/share/nginx/.lemme_try_hiding_flag_with_dot_in_front/flag_for_status.

    Nginx Alias Misconfiguration

    Excerpt Nginx configuration default.conf from wgmy-webtestonetwothree-frontend-conf ConfigMap:

          location /static {
        alias       /usr/share/nginx/html/;
        add_header  Cache-Control "private, max-age=3600";
      }

    This is vulnerable. For example, using /static../foo.txt, I can get the file /usr/share/nginx/foo.txt. Using this technique, I can obtain the flag for Status located at /usr/share/nginx/.lemme_try_hiding_flag_with_dot_in_front/flag_for_status.
